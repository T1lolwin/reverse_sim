# reverse_sim
# 拡散モデルのスコア関数と二重井戸ポテンシャルの可視化

修士研究で導出した偏微分方程式に基づく、スコアベース生成モデルのシミュレーションです。
C++ (Siv3D) を用いて、離散化されたスコア関数のベクトル場と粒子の時間発展をリアルタイム描画しています。

## 実行方法
1. Siv3D (v0.6.16) の空のプロジェクトを新規作成します。
2. このリポジトリの `Main.cpp` を上書きして配置します。
3. 以下のGoogle Driveリンクから `score_data.bin` をダウンロードし、プロジェクトの `App` フォルダ内に配置してください。
   👉 [ここにGoogle Driveの共有リンクを貼る]
4. Releaseビルドで実行してください。
